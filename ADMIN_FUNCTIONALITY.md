## 🔧 Technical Details

### State Management
All components use React `useState` hook for:
- Managing list of items (programs, courses, reports)
- Controlling modal visibility
- Tracking which item is being edited
- Managing form data

### Features
- **Add**: Opens modal with empty form → submit → adds to list
- **Edit**: Opens modal with pre-filled data → submit → updates item
- **Delete**: Confirmation dialog → removes from list
- **Search**: Real-time filtering (Courses only)
- **Modal**: Reusable form component with X button to close

### Form Validation
- All required fields marked with *
- Input fields with appropriate placeholders
- Numeric fields for credits/hours
- Dropdown selects for categories
- Auto-uppercase for codes

## ⚡ Performance Notes
- All CRUD operations are instant (client-side state)
- No backend API calls yet (local state management)
- Data resets on page refresh (as expected for local state)
