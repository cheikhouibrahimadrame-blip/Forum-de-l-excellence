## ✏️ CRUD Operations (All Pages)

### CREATE (Add New)
1. Click **+ Nouveau [Item]** button (top right)
2. Fill form fields (required marked with *)
3. Click **Créer** button
4. See item appear in list/grid

### READ (View)
- **Programs**: Grid cards
- **Courses**: Table with search
- **Reports**: Card grid
- Click on any item to see details

### UPDATE (Edit)
1. Click **pencil icon** on item
2. Modal opens with pre-filled form
3. Modify fields
4. Click **Mettre à jour**

### DELETE (Remove)
1. Click **trash icon** on item
2. Confirm in dialog: "Êtes-vous sûr?"
3. Item removed from list

## 📋 Form Fields Reference

### Programs Form
- **Name** (required): Full program name
- **Code** (required): Short code (auto-uppercase)
- **Department**: Department name
- **Level**: Education level
- **Duration**: How long (e.g., "1 an")

### Courses Form
- **Name** (required): Course name
- **Code** (required): Unique code
- **Program**: Which program (e.g., "CM2")
- **Teacher** (required): Teacher name
- **Credits**: Number of credits
- **Hours**: Hours per week
- **Semester**: T1, T2, or T3
- **Schedule**: Time slot (e.g., "Lun-Ven 8:00-9:00")

### Reports Form
- **Title** (required): Report name
- **Type** (required): Academic / Financial / Administrative
- **Department**: Department name
- **Generated By**: Person's name

## ❌ Common Issues & Fixes

| Issue | Fix |
|-------|-----|
| Page won't load | Check if `.\start-all.bat` is running |
| Can't login | Check email/password spelling |
| Changes don't save | Data is client-side only (page reload clears) |
| Search not working | Only works on Courses page |
| Modal won't open | Browser might be blocking popups |
| Button grayed out | Check form has all required fields (*) |
